Para ejecutar el proyecto, se debe abrir la carpeta "TP Final Objetos_Algoritmo Genetico/Main" y ejecutar usando Processing el archivo "Main.pde".

Una vez abierto el archivo, simplemente se ejecuta con el boton de "Play".

En caso de que se requiera iniciar una generación nueva a causa de vehículos incapaces de avanzar o morir, apriete la tecla "T".
